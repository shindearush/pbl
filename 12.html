Unlocking Insights: Descriptive Statistics with NumPy

By Code Daily

Data science is not just about moving numbers around; it is about summarizing them to find meaning. Before diving into complex machine learning models, one must understand the distribution of the data. Is it skewed? How much does it vary?

NumPy provides a suite of high-performance statistical functions that operate on arrays. In the example below, we calculate measures of central tendency (Mean, Median) and measures of spread (Variance, Standard Deviation).

We also explore the corrcoef function, which computes the Pearson correlation coefficient. This is essential for Feature Selection—determining which variables in your dataset are actually related to one another. 

import numpy as np

# Create a NumPy array
arr = np.array([10, 20, 30, 40, 50])

print("Array:", arr)

# Basic statistical calculations
print("\nMean:", np.mean(arr))
print("Median:", np.median(arr))
print("Standard Deviation:", np.std(arr))
print("Variance:", np.var(arr))

# Advanced statistical methods
print("\n25th Percentile:", np.percentile(arr, 25))
print("50th Percentile:", np.percentile(arr, 50))
print("75th Percentile:", np.percentile(arr, 75))

# Correlation coefficient
arr2 = np.array([5, 10, 15, 20, 25])
correlation = np.corrcoef(arr, arr2)

print("\nCorrelation Coefficient Matrix:")
print(correlation)

Note that np.corrcoef returns a matrix. The diagonal elements are always 1 (a variable's correlation with itself), while the off-diagonal elements show the correlation between the two different arrays.
