The Foundation of Data Science: Mastering NumPy Arrays

By Arush Shinde

In the world of data science, the standard Python list simply cuts it. When dealing with millions of data points, we need something faster, more compact, and more powerful. Enter NumPy (Numerical Python).

The ndarray object is the core of the library. It allows us to perform complex mathematical operations efficiently. In the script below, we explore four ways to create these arrays and how to inspect their metadata—such as their shape, data type, and size. 

import numpy as np

# Creating array using list
arr1 = np.array([1, 2, 3, 4, 5])

# Creating arrays using built-in functions
arr2 = np.zeros(5)
arr3 = np.ones(5)
arr4 = np.arange(1, 6)

# Display arrays
print("Array using list:", arr1)
print("Array of zeros:", arr2)
print("Array of ones:", arr3)
print("Array using arange:", arr4)

# Display attributes of array
print("\nAttributes of arr1:")
print("Shape:", arr1.shape)
print("Data type:", arr1.dtype)
print("Number of dimensions:", arr1.ndim)
print("Size:", arr1.size)

Understanding attributes like .shape and .ndim is crucial before moving on to matrix multiplication or reshaping data for machine learning models, as mismatched dimensions are the most common source of errors in data science.
