Beyond the Numbers: A Beginner's Guide to Matplotlib

By Code Daily

Numbers in a table can only tell you so much. To truly understand dataâ€”to spot the outliers, the trends, and the correlationsâ€”you need to visualize it. This is where Matplotlib shines. It is the grandfather of Python visualization libraries, offering granular control over every pixel of your chart.

The library operates on a hierarchy: you have the Figure (the canvas) and the Axes (the actual plot). In the code below, we explore four fundamental plot types:
1. Line Plot: Best for tracking changes over time.
2. Scatter Plot: Ideal for seeing how two variables relate to each other.
3. Histogram: Essential for seeing the distribution (frequency) of data.
4. Bar Plot: The go-to for comparing categorical data (like marks per student). 

import matplotlib.pyplot as plt

# Sample data
x = [1, 2, 3, 4, 5]
y = [10, 20, 15, 25, 30]

# ðŸ”¹ Line Plot
plt.figure()
plt.plot(x, y, color='blue', marker='o')
plt.title("Line Plot")
plt.xlabel("X-axis")
plt.ylabel("Y-axis")
plt.show()

# ðŸ”¹ Scatter Plot
plt.figure()
plt.scatter(x, y, color='red', marker='*')
plt.title("Scatter Plot")
plt.xlabel("X-axis")
plt.ylabel("Y-axis")
plt.show()

# ðŸ”¹ Histogram
data = [10, 20, 20, 30, 30, 30, 40, 50]
plt.figure()
plt.hist(data, color='green', edgecolor='black')
plt.title("Histogram")
plt.xlabel("Values")
plt.ylabel("Frequency")
plt.show()

# ðŸ”¹ Bar Plot
names = ["A", "B", "C", "D"]
marks = [70, 85, 60, 90]
plt.figure()
plt.bar(names, marks, color='purple')
plt.title("Bar Plot")
plt.xlabel("Students")
plt.ylabel("Marks")
plt.show()

Notice the call to plt.figure() before every plot. This creates a new "canvas" for each chart, ensuring they don't get drawn on top of each other. Once you have defined the data and labels, plt.show() renders the final image.
