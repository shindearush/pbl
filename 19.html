The Art of Comparison: Subplots and Statistical Charts

By Arush Shinde

Effective dashboards rarely rely on a single chart. To tell a complete story, you often need to place visualizations side-by-side: a trend line next to a distribution, or a summary next to the raw data. Matplotlib handles this via Subplots.

In this example, we create a layout with 1 row and 2 columns using plt.subplot(1, 2, index). This grid system allows us to render two distinct types of charts in a single figure:

1. Box and Whiskers Plot: A statistical powerhouse that summarizes data distribution. The box represents the interquartile range (where the middle 50% of data sits), while the whiskers extend to show variability.

2. Area Plot: Created using fill_between(), this emphasizes the magnitude of change over time, filling the space under the line to highlight volume. 

import matplotlib.pyplot as plt

# Sample data for box plot
marks = [45, 55, 60, 65, 70, 75, 80, 85, 90]

# Sample data for area plot
x = [1, 2, 3, 4, 5]
y = [2, 4, 6, 5, 7]

# Create a figure with 1 row and 2 columns
plt.figure(figsize=(10, 4))

# ðŸ”¹ Box and Whiskers Plot
# (1 row, 2 cols, index 1)
plt.subplot(1, 2, 1)
plt.boxplot(marks)
plt.title("Box and Whiskers Plot")
plt.ylabel("Marks")

# ðŸ”¹ Area Plot
# (1 row, 2 cols, index 2)
plt.subplot(1, 2, 2)
plt.fill_between(x, y, color="skyblue")
plt.title("Area Plot")
plt.xlabel("X values")
plt.ylabel("Y values")

# Display the figure
plt.show()

Using figsize=(10, 4) is crucial here; without it, the two plots might look squashed. This sets the width to 10 inches and height to 4 inches, ensuring clear readability for both visualizations.
