The Split-Apply-Combine Strategy: Mastering Pandas GroupBy

By Arush Shinde

Raw data is often granular—individual sales, daily logs, or single employee records. However, business insights usually come from summaries: What is the total revenue per region? What is the average salary per department? This is where the Split-Apply-Combine pattern comes in.

Pandas handles this elegantly with the groupby() method. It conceptually splits the data into buckets based on a key (in this case, "Department"), applies a function (like summing values), and then combines the results back into a new DataFrame.

In the code below, we go a step further using the .agg() method, which allows us to calculate multiple statistics—both the total (sum) and the average (mean)—in a single line of code. 

import pandas as pd

# Create DataFrame
data = {
    "Department": ["IT", "IT", "HR", "HR", "Sales", "Sales"],
    "Employee": ["A", "B", "C", "D", "E", "F"],
    "Salary": [50000, 55000, 40000, 42000, 45000, 48000]
}

df = pd.DataFrame(data)

print("Original DataFrame:")
print(df)

# Group by Department and calculate sum and mean of Salary
grouped = df.groupby("Department")["Salary"].agg(["sum", "mean"])

print("\nGrouped Data with Sum and Mean:")
print(grouped)

The resulting grouped DataFrame uses "Department" as its index. This transformation is powerful because it turns a list of individual records into a concise summary table ready for reporting or visualization.
