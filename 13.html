Data at Your Fingertips: Getting Started with Pandas DataFrames

By Arush Shinde

While NumPy handles the heavy mathematical lifting, Pandas provides the structure. It introduces the DataFrame—a table-like structure that organizes data into rows and columns, similar to an Excel spreadsheet or a SQL table, but with the full power of Python behind it.

In this workflow, we load raw data from a CSV file, inspect its structure, and perform a simple calculation. Notice the function df.info(); this is your first stop when loading any dataset. It reveals missing values (nulls) and data types (integers, floats, strings), helping you catch messy data before you begin analysis. 

import pandas as pd

# Read data from CSV file
df = pd.read_csv("data.csv")

# Display first few rows
print("First 5 rows of DataFrame:")
print(df.head())

# Display information about DataFrame
print("\nDataFrame Information:")
print(df.info())

# Display basic statistics
print("\nBasic Statistics:")
print(df.describe())

# Modify DataFrame (example: add new column)
# axis=1 ensures we sum across columns (horizontally)
df["Total"] = df.sum(axis=1, numeric_only=True)

# Save modified DataFrame to new CSV file
df.to_csv("modified_data.csv", index=False)

print("\nModified DataFrame saved successfully.")

We also used df.describe(), which instantly generates a statistical summary (mean, min, max, quartiles) for every numerical column. Finally, we saved our work back to a new CSV, setting index=False to avoid creating an annoying extra column of row numbers in our output file.
