From Scratch: Creating and Inspecting Pandas DataFrames

By Arush Shinde

While data often arrives in CSV or Excel files, sometimes you need to construct a dataset manually—perhaps for unit testing, creating small lookup tables, or recording experimental results. The most intuitive way to do this is by converting a Python Dictionary into a DataFrame.

In the code below, observe how the dictionary keys (e.g., "Name", "Marks") automatically become the column headers, while the lists of values become the rows. Once the DataFrame is created, we use head() and tail() to peek at the data without overwhelming our console, and describe() to get a quick statistical snapshot. 

import pandas as pd

# Create DataFrame from dictionary of lists
data = {
    "Name": ["Arush", "Riya", "Amit", "Neha", "Rahul"],
    "Marks": [85, 90, 78, 88, 92],
    "Age": [20, 21, 19, 20, 22]
}

df = pd.DataFrame(data)

# Display DataFrame
print("DataFrame:")
print(df)

# Display first few rows
print("\nFirst 3 rows:")
print(df.head(3))

# Display last few rows
print("\nLast 3 rows:")
print(df.tail(3))

# Display information about DataFrame
print("\nDataFrame Information:")
print(df.info())

# Display statistical summary
print("\nStatistical Summary:")
print(df.describe())

The describe() method is particularly useful for sanity checks. For instance, if the "Age" column showed a max value of 200, or the "Marks" count was lower than the "Name" count, you would immediately know your data has errors or missing values.
