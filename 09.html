Dimensionality Mastery: Reshaping and Transposing Data with NumPy

By Arush Shinde

Data rarely arrives in the exact format we need. You might receive a flat list of pixels that needs to become a 2D image, or a time-series dataset that needs pivoting. NumPy excels at this structural manipulation without copying the underlying data, making it incredibly fast.

In this guide, we start with a simple 1D array and transform it into a matrix (2x3). We then explore the Transpose operation—flipping rows into columns—and finally dive into 3D space by swapping axes, a common operation when preparing image data for Neural Networks (e.g., switching from Channel-Last to Channel-First formats). 

import numpy as np

# Create a NumPy array
arr = np.array([1, 2, 3, 4, 5, 6])

print("Original Array:")
print(arr)

# Reshape into 2x3 matrix
reshaped = arr.reshape(2, 3)
print("\nReshaped Array (2x3):")
print(reshaped)

# Transpose of reshaped array
transpose_arr = reshaped.T
print("\nTranspose of Reshaped Array:")
print(transpose_arr)

# Create 3D array
arr3d = np.arange(1, 9).reshape(2, 2, 2)
print("\n3D Array:")
print(arr3d)

# Swap axes
swapped = np.swapaxes(arr3d, 0, 1)
print("\nArray after Swapping Axes (0 and 1):")
print(swapped)

The .reshape() method is versatile but strict: the total number of elements must remain constant. Trying to reshape an array of 6 elements into a 2x4 matrix (requiring 8 slots) will result in a ValueError.
