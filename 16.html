Finding the Signal: Filtering and Sorting Data with Pandas

By Code Daily

Analyzing data often feels like searching for a needle in a haystack. You rarely need the entire dataset; instead, you need to extract specific rows that meet certain criteriaâ€”like high-performing students, recent transactions, or critical errors.

In this guide, we explore two ways to filter data. The first is Boolean Indexing, the classic approach where we pass a condition (True/False mask) to the DataFrame. The second is the query() method, which allows for cleaner, SQL-like syntax that is often easier to read.

Finally, we demonstrate sort_values(). Data is most useful when orderedâ€”whether you are creating a leaderboard or organizing events chronologically. 

import pandas as pd

# Create DataFrame
data = {
    "Name": ["Arush", "Riya", "Amit", "Neha", "Rahul"],
    "Marks": [85, 90, 72, 88, 95],
    "Age": [20, 21, 19, 20, 22]
}

df = pd.DataFrame(data)

print("Original DataFrame:")
print(df)

# ðŸ”¹ Filtering using Boolean Indexing
filtered_df = df[df["Marks"] > 85]
print("\nStudents with Marks > 85:")
print(filtered_df)

# ðŸ”¹ Filtering using query() method
query_df = df.query("Marks > 85")
print("\nFiltered using query():")
print(query_df)

# ðŸ”¹ Sorting the filtered DataFrame by Marks
sorted_df = filtered_df.sort_values(by="Marks", ascending=False)
print("\nSorted DataFrame by Marks (Descending):")
print(sorted_df)

While df[df["Col"] > 5] is powerful, the code can get messy with multiple conditions (e.g., using & for AND, | for OR). In those complex cases, df.query("Col > 5 and Age < 30") often shines by keeping your code readable and expressive.
